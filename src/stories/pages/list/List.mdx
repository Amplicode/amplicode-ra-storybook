# List

The `<List>` component is the root component for list pages. It fetches a list of records from the data provider, puts it in a ListContext, renders the default list page layout (title, buttons, filters, pagination), and renders its children. Usual children of `<List>`, like `<Datagrid>`, are responsible for displaying the list of records. [More information](https://marmelab.com/react-admin/List.html).
